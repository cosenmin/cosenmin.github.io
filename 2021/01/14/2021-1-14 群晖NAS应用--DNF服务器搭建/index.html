一、群晖NAS中安装 centos5.8

1.安装Virtual Machine Manager

打开群晖套件中心，搜索’vir’，安装Virtual Machine Manager套件。

2.安装映像文件

打开Virtual Machine Manager套件，配置好后点击映像->新增，将你的iso文件添加到映像文件中。

然后在虚拟机中点击新增，选中Linux

根据你的内存大小配置，一般2h2g差不多

配置虚拟硬盘大小，一般30G够用

网络直接默认，其他设置里面启动ios文件选择4G那个，其他iso文件不用管，自动自动设置为是。

然后是选择一个账户

接着直接默认然后点击应用就可以了

3.安装centos 5.8

安装完成后，先不着急开机，如果已经开机，可点击关机或者强制关机让它进入关机状态，紧接着选中虚拟机，点击操作->编辑->存储->设置

然后将虚拟盘控制器设置为IDE控制器。

设置完成后，可以点击连接即可启动centos5.8系统，初次启动会进入centos5.8的安装步骤。

这个页面直接按Enter过

这里直接选择Skip按空格跳过

然后进入安装界面

首先是语言，选择简体中文，然后软键盘选择美国英语式，点下一步的时候选择是

这里选择的是最下面的用户自定义分区，注：不是图片这个！！！选择然后进入分区的挂载，其挂载的大小大概如下分配（这里笔者暂时没找到图，只能口述,其实最重要的是/和swap，另外两个好像可以不要）

‘/‘—20000

‘home’—5000

‘swap’—4000

‘boot’—200

紧接着是选择网络、选择时区、设置root密码，这些都直接下一步。然后来到图形界面设置，类似下图，不过这里笔者用winSCP和putty进行连接和文件操作，就不需要图形界面，所以用箭头那四个选项都不要选！！！

然后等待安装完成重启即可使用centos5.8了

二、安装DNF服务端

1.使用putty和winscp连接你的centos

使用网页连接进入centos，输入密码后可输入命令ifconfig来查看centos在局域网中的ip地址。

然后为了避免不必要的麻烦，我们可以通过putty和winscp连接centos5.8来进行命令输入和文件传输。

首先，putty运行一下命令关闭防火墙：

chkconfig iptables off
service iptables stop
setenforce 0

2.替换CentOS-Base.repo文件

由于centos5版本太旧，原本网易的镜像库都已经失效了，因此需要修改CentOS-Base.repo文件，在文件夹repo里面，有分别国外和清华两个选项，这里最好选清华，因为下载很快。

打开winscp，进入到/etc/yum.repos.d目录下，替换CentOS-Base.repo。

3.启动一键安装服务端

将一键端的5个文件放在根目录

putty运行命令:

cd /;chmod -R 0777 /install2;./install2

运行后首先会下载国外的源，30~90分钟不等，然后开始根据提示安装即可。

4.检查ip是否为外网ip

首先找到/home/neople/channel/cfg下的channel.cfg，将this_ip改为你的外网ip。

然后找到/home/neople/game/cfg里面的siroco01.cfg、siroco11.cfg（游戏频道）和siroco52.cfg（决斗场频道），将里面的如下选项改为你的外网ip。

ip
udp_ip_of_hades
ipg_ip
nxj_ipg_ip
relay_ip
stun_ip
stun_ip
stun_ip

然后在路由器里面映射如下端口

7000-7001
10006-10031
31003
9006
7200
2311-2313
11006-11031
10315
11315

5.上传pvf、等级补丁和公钥到

上传目录：/home/neople/game

pvf–游戏的版本文件

等级补丁–角色最高等级

公钥–登录器识别（这里用的是统一登录器）

上传完成后putty运行命令:

cd /root
./run 启动服务器（几分钟）

等到跑出四国为止。若想关闭服务器，只需输入两次./stop即可。

6.统一登录器网关配置

首先，需要上传libmysqlclient.so.15到 usr/lib64下。

然后啊，将网关配置文件放到root/下，右键设置权限为0777。

完成后，只需要输入./GateRestart重启网关即可。

Config.ini里面的ConnectKey必须与统一登录器的通讯密钥一致。

三、客户端配置

1.安装客户端

解压那四个压缩包到dxf70目录即可

2.配置登录器

首先，将统一登录器5.4客户端补丁中的所有文件覆盖到dxf70目录下。

然后，打开统一网关在线管理工具v5.4，在游戏地址和网关地址上填上你的公网ip，然后点击生成登录器，并将生成的登录放到dxf70目录下。

注意：网关设置的通信密钥和登录器设置里面的通信密钥要相同。

然后在网关设置->网关地址里面填上你的centos的局域网ip，其他不变，然后点击连接。

连接之后点击实用工具->打开文件，选择你的pvf进行，点击获取当前密码->生成随机密码->设置密码生效进行加密，最后点击文件加密生成一个对应加密的pvf，将这个pvf放到dxf70目录。

为什么不能直接放pvf？因为统一登录器客户端直接放pvf会启动失败，必须放加密的。

完成后运行统一登录器，注册账号后点击进入游戏即可开始愉快地玩耍了。

注意：win10直接点击登录，win7要右键管理员运行登录器。

四、数据库配置

如果出现能进去游戏也能玩，但就是GM工具怎么也连接不上的只能通过Navicat的ssh才能连接上，如下图所示：

那么，解决方法是进入lampp的mysql数据库（一键端自带的mysql），新增一个授权了本机ip的账户，在本机上用该账户登录即可，具体步骤如下

首先，利用winscp进入192.168.101.110虚拟机，找到文件/opt/lampp/etc/my.cnf，在[mysqld]下面添加一行skip-grant-tables内容如图：（表示不用密码登录数据库，设置好后删除即可）

修改完后输入/opt/lampp/lampp restart进行mysql的重启。

然后输入命令/opt/lampp/bin/mysql -ugame -p 登录你的game账户（如果第1步添加了代码，这里直接按enter就能进去，没添加就要输入uu5!^%jg进去）

接着依次输入以下代码可以查看数据库的内容、user、host表

    use mysql;
    show databases;
    select host,user from user;

输入 flush privileges;刷新权限表

输入grant all privileges on . to ‘新用户名‘@’需要连接电脑的ip’ identified by ‘新密码’ with grant option;创建一个新的用户名密码账户

然后输入select host,user from user;查看

账户算是创建好了，接着可以回到第一步将skip-grant-tables注释掉或者删除（毕竟没密码不安全）
最后ctrl+c退出mysql重启
/opt/lampp/lampp restart

然后用新账号密码完美登录

常见问题解决

1.win10无法组队的问题。

下载一个uu加速器，选择加速dnf，将两台电脑设置到统一节点即可组队。

2.客户端点进入游戏会闪退。

win7的话把所有杀毒软件都关了，win10的话左下角搜索’病毒和威胁防护’->病毒和威胁防护设置->管理设置->关闭实时防护，或者排除项->添加或删除排除项,将整个客户端文件包括进去即可。

3.开启拍卖行的方法

打开Navicat连接数据库，进入表xxx中，复制xxx和xxx两份表（仅复制结构），然后更改日期和系统日期相同。

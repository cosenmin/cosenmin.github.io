如何在Github上搭建个人博客–基础篇

一、安装yilia主题

安装主题很简单，只需要在博客根目录下右键打开Git Bash，然后输入以下代码即可安装。

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

安装完成后你的themes文件夹里会多出一个yilia目录，这就是主题目录了，下面对目录中的一些需要用到的文件或文件夹做简单的解释：

_config.yml                主题配置
\layout                    主题布局目录
    categories.ejs            分类界面布局
    tag.ejs                标签界面布局
    archive                归档界面布局
    \_partial
        article.ejs        文章布局
        footer.ejs        页脚布局
        left-col.ejs        个人信息界面布局
        \post
            word.ejs    自定义的文章布局
\source                    主题源码目录

了解之后下面就可以开始主题编辑了，每次编辑后如果本地预览的话可以在博客根目录的Git Bash里面输入以下代码，然后即可在浏览器中输入http://localhost:4000/查看，看完后记得ctrl + c取消。

 (hexo clean;hexo g;hexo s)

如果要部署到github或者gitee上，可以输入以下代码进行部署：

 (hexo clean;hexo g;hexo d)

二、整体配置

1.名字、签名

打开主题配置文件，在# Header和menu:之间插入以下代码：

author: 作者名字
subtitle: 作者签名

2.头像、网页图标

准备好一张.png的图片，不要太大，几十k就可以了。

然后在博客根目录下创建文件夹source\assets\blogImg，将你的头像放进去。

接着在你的主题配置文件里面搜索avatar，将你的路径和图片放进去，如下所示：

#注意: 冒号后面留一个空格
avatar:  /assets/blogImg/图片名字.png

图标也是同理，不过图标文件一般是16x16或者32x32的，记得修改大小，代码如下所示:

#注意: 冒号后面留一个空格
favicon:  /assets/img/图标名字.png

3.在页尾显示博客的访问总量、访问数

首先，要实现计数功能，要添加卜算子计数插件，打开页脚布局文件footer.ejs，在最后添加以下一行代码：

<script async src="//busuanzi.ibruce.info/busuanzi/2.3/busuanzi.pure.mini.js"></script>

然后继续在footer.ejs文件中，在其中间添加以下两行代码：

<span id="busuanzi_container_site_pv"> 总访问量: <span id="busuanzi_value_site_pv"></span> /</span>
<span id="busuanzi_container_site_uv"> 访客数:<span id="busuanzi_value_site_uv"></span></span>

完成后重新部署即可，效果展示：

本地查看的时候访问量可能显示异常，别担心，放到远程仓库就好了。

二、文章配置

1.字数统计、阅读时长、阅读量

由于前面已经填了卜算子，这里就不需要再添加了。

首先，在\layout_partial\post文件夹中新建一个自定义文章布局文件word.ejs，在其中添加如下代码:

<div style="margin-top:10px;">
<span class="post-time">
  <span class="post-meta-item-icon">
<i class="fa fa-keyboard-o"></i>
<span class="post-meta-item-text" style="font-size: 16px;color: grey">  字数统计: </span>
<!--这里样式可以自己定制-->
<span class="post-count" style="font-size: 16px;color: grey"><%= wordcount(post.content) %>字</span>
  </span>
</span>

<span class="post-time">
  &nbsp; | &nbsp;
  <span class="post-meta-item-icon">
<i class="fa fa-hourglass-half"></i>
<span class="post-meta-item-text" style="font-size: 16px;color: grey">  阅读时长: </span>
<span class="post-count" style="font-size: 16px;color: grey"><%= min2read(post.content) %>分</span>
  </span>
</span>
</div>

然后直接打开文章布局文件article.ejs，在其中添加代码如下所示：

 <header class="article-header">
<%- partial('post/title', {class_name: 'article-title'}) %>
<% if (!post.noDate){ %>
<%- partial('post/date', {class_name: 'archive-article-date', date_format: null}) %>
         
         <!-- 开始添加字数统计-->
<%- partial('post/word') %>
<!-- 添加完成 -->
        
         <% } %>
         
        <!-- 开始添加字数统计2-->
         
        <!-- 添加代码 -->
<% if ( !index ){ %>
  <span class="archive-article-date">
  阅读量 <span id="busuanzi_value_page_pv"></span>
  </span>
<% } %>
        
  </header>

至此，字数统计、阅读时长、阅读量就弄好了，赶紧部署到本地预览一下吧。

2.文章分类的实现

如果我们坚持写下去的话文章就会越来越多，希望对某一类文章进行分类，这时可以在菜单栏添加一个分类的选项，并对其进行布局，具体步骤如下：

(1)打开主题配置文件，在menu栏下加上如下内容

menu:
  主页: /home
  分类: /categories

(2)打开Git Bash，输入以下代码创建categories配置文件

hexo new page categories

(3)打开\source\categories\index.md文件，修改如下：

---
title: 分类
date: 2021-01-7 18:18:25
toc: false
comments: false
layout: categories
---

(4)修改或覆盖\themes\yilia\layout\categories.ejs文件如下：

<article class="article article-type-post show">
  <header class="article-header" style="border-bottom: 1px solid #ccc">
  <h1 class="article-title" itemprop="name">
<%= page.title %>
  </h1>
  </header>

  <% if (site.categories.length){ %>
  <div class="category-all-page">
<h3>说明：给文章简单地做一下分类</h3>
<h3>共计&nbsp;<%= site.categories.length %>&nbsp;个分类</h3>
<%- list_categories(site.categories, {
  show_count: true,
  class: 'category-list-item',
  style: 'list',
  depth:3,   
  separator: ''
}) %>
  </div>
  <% } %>
</article>

(5)添加如下代码到themes\yilia\source\main.0cf68a.css中：

category-all-page {
margin: 30px 40px 30px 40px;
position: relative;
min-height: 70vh;
  }
  .category-all-page h2 {
margin: 20px 0;
  }
  .category-all-page .category-all-title {
text-align: center;
  }
  .category-all-page .category-all {
margin-top: 20px;
  }
  .category-all-page .category-list {
margin: 0;
padding: 0;
list-style: none;
  }
  .category-all-page .category-list-item-list-item {
margin: 10px 15px;
  }
  .category-all-page .category-list-item-list-count {
color: $grey;
  }
  .category-all-page .category-list-item-list-count:before {
display: inline;
content: " (";
  }
  .category-all-page .category-list-item-list-count:after {
display: inline;
content: ") ";
  }
  .category-all-page .category-list-item {
margin: 10px 10px;
  }
  .category-all-page .category-list-count {
color: $grey;
  }
  .category-all-page .category-list-count:before {
display: inline;
content: " (";
  }
  .category-all-page .category-list-count:after {
display: inline;
content: ") ";
  }
  .category-all-page .category-list-child {
padding-left: 10px;
  }

至此，文章分类添加完毕，当修改或者新写一篇文章的时候，可以在文章中添加一行代码即可。以如下示例，就是将这篇文章归类到了一个”个人博客”的分类中，可以在菜单栏点击分类查看。

---
title: 如何在Github上搭建个人博客--基础篇
categories:
- 个人博客
---

3.文章标签的实现

当你写文章时，可以根据文章的方向、题材、内容确定一个或者多个标签，当我们需要查看某一类标签的文章时，就可以通过标签迅速查看。文章标签的实现和分类的实现差不多，这里就不过多阐述。

(1)打开主题配置文件，在menu栏下加上如下内容

menu:
  主页: /home
  标签: /tags
  分类: /categories

(2)打开Git Bash，输入以下代码创建tags配置文件

hexo new page tags

(3)打开\source\categories\index.md文件，修改如下：

---
title: 标签
date: 2021-01-06 11:55:27
type: tags
comments: false
layout: tag
---

(4)修改或覆盖\themes\yilia\layout\tag.ejs文件如下：

<article class="article article-type-post show">
  <header class="article-header" style="border-bottom: 1px solid #ccc">
  <h1 class="article-title" itemprop="name">
<%= page.title %>
  </h1>
  </header>
  
  <h3>共计&nbsp;<%= site.tags.length %>&nbsp;个标签</h3>
  <div class="tag-cloud">
<%- tagcloud({
  min_font: 20, 
  max_font: 35, 
  amount: 999
}) %>
  </div>
</article>

(5)添加如下代码到themes\yilia\source\main.0cf68a.css中：

.tag-cloud {
    text-align:center;
    padding:20px
}
.tag-cloud a {
    margin:0 20px;
    word-break:keep-all;
    position:relative
}
.tag-cloud a:hover::before {
    width:100%;
    left:0;
    right:auto
}
.tag-cloud a::before {
    content:"";
    position:absolute;
    width:0;
    right:0;
    bottom:0;
    height:2px;
    background:#08c;
    -webkit-transition:width .3s ease;
    transition:width .3s ease
}

4.文章归档的实现

归档，其实就是把你写的所有文章按照日期进行排序，相比于之前两个，归档就比较简单了，只需要前三步就可以了。

(1)打开主题配置文件，在menu栏下加上如下内容

menu:
  主页: /home
  标签: /tags
  分类: /categories
  归档: /archives

(2)打开Git Bash，输入以下代码创建/archives配置文件

hexo new page /archives

(3)打开\source\archives\index.md文件，修改如下：

---
title: 归档
date: 2021-01-07 09:37:47
toc: false
comments: false
layout: archive
---

效果展示：

至此，菜单栏算是布置完成了。

5.文章、菜单栏插入图标

菜单栏布置完成了，但笔者感觉仍然是光秃秃的，如果在主页前面有个这样的图标就好了。

首先，需要下载图标库，这里选择的是fontawesome图标库，这里为了以后方便，笔者在淘宝上花了20大洋购买了其的本地图标库，下载好后将除了svgs的其他文件夹解压到themes\yilia\source目录(由于svgs太多图标，因此这里我只选了一些博客需要用到的图标传过去)。

fontawesome私有云下载

下载好后有两个压缩包，fa…压缩包里面主要放的本地图标文件，图标代码PDF主要是方便查找检索图标。

接着需要在themes\yilia\layout_partial\head.js文件中添加以下带来来导入fontawesome的图标配置。

<link rel="stylesheet" href="/css/font-awesome.css">

在一般文章中插入图标只需要输入以下代码即可:

<i class="fas fa-图标名">

而在菜单栏添加图标还需要修改D:\Blog\themes\yilia\layout_partial\left-col.ejs文件，代码所示如下：

<nav class="header-menu">
        <ul>
            <% for (var i in theme.menu){ %>
                <%if (i === "主页" ){%>
    					
  • <%= i %>
    • 
    				<%}else if (i === "标签" ){%>
    					
  • <%= i %>
    • 
    				<%}else if (i === "分类" ){%>
    					
  • <%= i %>
    • 
    				<%}else if (i === "归档" ){%>
    					
  • <%= i %>
    • 
    				<%}else{%>
    				<% continue %>
    				<%}%>
            <%}%>
            </ul>
        </nav>

    如此，即可将图标添加到菜单栏前面了。
